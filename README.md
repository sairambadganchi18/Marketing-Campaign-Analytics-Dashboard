# 📊 Marketing Campaign Analytics Dashboard

## 📌 Project Overview

The **Marketing Campaign Analytics Dashboard** is an end-to-end Business Intelligence project developed in **Power BI** to analyze and monitor marketing campaign performance across multiple digital marketing channels.

The dashboard provides actionable insights into campaign effectiveness, customer engagement, marketing spend, and overall business performance. It helps stakeholders make data-driven decisions by tracking key marketing KPIs and identifying top-performing campaigns and channels.

---

## 🎯 Business Objective

The objective of this project is to:

- Analyze marketing campaign performance.
- Measure Return on Investment (ROI).
- Compare marketing channels.
- Track customer engagement.
- Optimize marketing spend.
- Improve campaign decision-making using interactive dashboards.

---

## 🏗️ Data Model

This project follows a **Star Schema** data model.

### Fact Table
- Fact_CampaignPerformance

### Dimension Tables
- Dim_Date
- Dim_Campaigns
- Dim_Channels
- Dim_Customers

## 📈 Dashboards Included

### 🏠 Landing Page
Interactive navigation page for easy dashboard access.

### 📊 Executive KPI Dashboard
Displays high-level business metrics including:
- Total Revenue
- Total Spend
- ROI %
- Total Impressions
- Total Clicks
- Total Conversions

---

### 📌 Campaign Overview Dashboard

Provides campaign-level analysis including:

- Campaign Performance
- Revenue by Campaign
- Spend by Campaign
- Conversion Rate
- Customer Acquisition Cost (CAC)
- Engagement Rate
- Top Performing Channel

---

### 📢 Channel Performance Dashboard

Analyzes performance across different marketing channels.

Channels Included:

- Google Ads
- Facebook Ads
- Instagram Ads
- Email
- SMS
- Website
- Branch

Metrics:

- Revenue vs Spend
- CTR %
- Average Revenue per Channel
- Highest CTR
- Channel-wise Performance

---

### 👥 Customer Engagement Dashboard

Provides customer behavior insights.

Analysis includes:

- Conversion by Gender
- Conversion by Region
- Conversion by Customer Segment
- Income Bracket Analysis
- Age Group Analysis
- Customer Engagement
